Explanation

Because...you know...you might need it

This website is a bit different from other websites. What this means for you is that you're not able to scroll up and down like you're used to, but you will have to use the navigation bar on the left side of the screen instead.

The control are simple: just use the arrow keys to move the snek in four directions. However, take caution not to hit the walls or accidentally eat yourself, otherwise it's game over for you!

Make sure to click the "Game" button first!
